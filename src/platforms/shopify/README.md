# Shopify Platform Adapter for Migrate to Woo CLI

This directory contains the specific components required to migrate product data from Shopify using the `migrate-to-woo-cli` plugin.

## Files

*   `class-shopify-fetcher.php`: Implements `Platform_Fetcher_Interface`. Handles communication with the Shopify GraphQL API to fetch product data batches.
*   `class-shopify-mapper.php`: Implements `Platform_Mapper_Interface`. Transforms the raw product data received from the Shopify API into the standardized array format expected by the core `WooCommerce_Product_Importer`.
*   `config-example.php`: Example configuration file. Copy this to `config.php` and add your Shopify API credentials.
*   `config.php` (You create this): Contains your specific Shopify `SHOPIFY_DOMAIN` and `ACCESS_TOKEN` constants used by the Fetcher.

## Field Mapping (`Shopify_Mapper`)

The `Shopify_Mapper` class takes the product node object from the Shopify GraphQL API (`SHOPIFY_PRODUCT_QUERY` in the Fetcher) and maps its fields to the following standardized keys in the `$wc_data` array, which is then passed to the `WooCommerce_Product_Importer`:

| Standardized `$wc_data` Key | Shopify GraphQL Source Field(s)                                                                                                                                                                | Notes                                                                                                                  |
| :-------------------------- | :--------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- | :--------------------------------------------------------------------------------------------------------------------- |
| `original_product_id`       | `id` (basename)                                                                                                                                                                                | The numeric Shopify Product ID.                                                                                        |
| `is_variable`               | Checks if `variants.edges` count > 1                                                                                                                                                           | Boolean flag.                                                                                                          |
| `name`                      | `title`                                                                                                                                                                                        |                                                                                                                        |
| `slug`                      | `handle`                                                                                                                                                                                       |                                                                                                                        |
| `description`               | `descriptionHtml`                                                                                                                                                                              | Raw HTML from Shopify.                                                                                                 |
| `status`                    | `status`                                                                                                                                                                                       | Maps `ACTIVE` to `publish`, others likely to `draft`.                                                                  |
| `date_created_gmt`        | `createdAt`                                                                                                                                                                                    | Assumed to be in UTC/GMT.                                                                                              |
| `catalog_visibility`      | `onlineStoreUrl`                                                                                                                                                                               | Maps `null` to `hidden`, otherwise `visible`.                                                                          |
| `original_url`            | `onlineStoreUrl`                                                                                                                                                                               | The storefront URL if the product is visible.                                                                          |
| `categories`                | `collections.edges` -> `node.title`, `node.handle`                                                                                                                                             | Array of `['name' => ..., 'slug' => ...]`.                                                                             |
| `tags`                      | `tags` (array)                                                                                                                                                                                 | Array of `['name' => ..., 'slug' => ...]`.                                                                             |
| `brand`                     | `vendor`                                                                                                                                                                                       | Array `['name' => ..., 'slug' => ...]` if vendor exists.                                                               |
| `images`                    | `media.edges` -> `node.id`, `node.image.url`, `node.image.altText` & `featuredMedia.id` check                                                                                                  | Array of `['original_id' => ..., 'url' => ..., 'alt' => ..., 'is_featured' => bool]`.                                  |
| `metafields`                | `metafields.edges` -> `node.namespace`, `node.key`, `node.value`                                                                                                                               | Associative array `['{$namespace}_{$key}' => $value]`. Example: `global_title_tag`.                                    |
| `attributes` (Variable)     | `options` -> `name`, `values`, `position`                                                                                                                                                      | Array of `['name' => ..., 'options' => [...], 'position' => ..., 'is_visible' => true, 'is_variation' => true]`.        |
| `variations` (Variable)     | `variants.edges` -> `node` (See details below)                                                                                                                                                 | Array of associative arrays, each representing a variation.                                                            |
| `original_variant_id` (Simple) | `variants.edges[0].node.id` (basename)                                                                                                                                                       | Original ID of the *first* variant for simple products.                                                                |
| `regular_price` (Simple)    | `variants.edges[0].node.price` or `compareAtPrice`                                                                                                                                             | From the *first* variant.                                                                                              |
| `sale_price` (Simple)       | `variants.edges[0].node.price` (if `compareAtPrice` set & higher)                                                                                                                              | From the *first* variant.                                                                                              |
| `sku` (Simple)              | `variants.edges[0].node.sku`                                                                                                                                                                   | From the *first* variant.                                                                                              |
| `manage_stock` (Simple)     | `variants.edges[0].node.inventoryItem.tracked`                                                                                                                                                 | From the *first* variant.                                                                                              |
| `stock_status` (Simple)     | Based on `inventoryQuantity` & `inventoryPolicy`                                                                                                                                               | From the *first* variant. (`instock` / `outofstock`).                                                                    |
| `stock_quantity` (Simple)   | `variants.edges[0].node.inventoryQuantity`                                                                                                                                                     | From the *first* variant.                                                                                              |
| `weight` (Simple)           | `variants.edges[0].node.inventoryItem.measurement.weight` (converted)                                                                                                                        | From the *first* variant. Converted to store's unit.                                                                   |

### Variation Mapping Details

Each element in the `variations` array (for variable products) is an associative array with these keys, mapped from the `variants.edges.node` object:

| `$wc_data['variations'][N]` Key | Shopify GraphQL `variants.edges.node` Source Field(s)                                     | Notes                                                    |
| :----------------------------- | :---------------------------------------------------------------------------------------- | :------------------------------------------------------- |
| `original_id`                  | `id` (basename)                                                                           |                                                          |
| `regular_price`                | `price` or `compareAtPrice`                                                               | `compareAtPrice` used if higher than `price`.            |
| `sale_price`                   | `price` (if `compareAtPrice` set & higher)                                                | `null` otherwise.                                        |
| `sku`                          | `sku`                                                                                     |                                                          |
| `manage_stock`                 | `inventoryItem.tracked`                                                                   | Boolean.                                                 |
| `stock_status`                 | Based on `inventoryQuantity` & `inventoryPolicy`                                          | `instock` / `outofstock`.                                |
| `stock_quantity`               | `inventoryQuantity`                                                                       |                                                          |
| `weight`                       | `inventoryItem.measurement.weight` (value + unit)                                         | Converted to store's weight unit.                        |
| `attributes`                   | `selectedOptions` -> `name`, `value`                                                      | Associative array `[$name => $value]` (e.g., `['Color' => 'Blue']`). |
| `image_original_id`            | `media.edges[0].node.id`                                                                  | Original ID of the variant's assigned MediaImage.        |
| `menu_order`                   | `position`                                                                                |                                                          |
